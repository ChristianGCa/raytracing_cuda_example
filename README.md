# Example of Ray Tracing using CUDA

![Output frames](./animation.gif)

To execute:
```bash
make
```
To run, go to the build folder:
```bash
cd build
```
Ray Tracing with CUDA:
```bash
./raytrace_cuda
```
Ray Tracing with CPU:
```bash
./raytrace_cpu
```
Frames Ray Tracing with CPU:
```bash
./raytrace_cpu_frames
```
